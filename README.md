# NoteTube - YouTube Notes Genius

Transform YouTube videos into comprehensive study notes with AI.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS (Midnight Aurora Theme)
- shadcn-ui
- Supabase / Custom Backend

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Features

- **AI-Powered Notes**: Automatically generate structured notes from any YouTube video.
- **Midnight Aurora Theme**: A premium, dark-mode-first aesthetic with glassmorphism.
- **History Tracking**: Keep track of all your processed videos.
- **Export Options**: Copy to clipboard, download as TXT, or Print/PDF.
